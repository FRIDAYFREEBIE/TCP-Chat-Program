# TCP Chat Program (Client–Server)

### 🎮 세번째 토이 프로젝트

---

## 📌 **프로젝트 개요**
- **프로젝트 유형**: 콘솔 기반 채팅 프로그램
- **네트워크 모델**: TCP Client-Server 구조
- **팀 구성**: 1인 개발
- **목표**: C# 네트워크 프로그래밍 이해, TCP 통신 구조 학습, 멀티스레드 환경에서의 클라이언트 처리 방식 이해

---

## 🔑 **주요 기술 사항**
- **언어**: C#
- **멀티스레딩**: 클라이언트에 각자의 스레드를 할당합니다. 메시지 송신과 수신에 사용되는 스레드를 두 개로 나누었습니다. 따라서 메시지 송수신을 동시에 처리할 수 있습니다.
- 

---

## 🤔 **느낀점과 배운점**
- **TCP 통신의 흐름(Client → Server → Client)** 을 명확히 이해할 수 있었습니다.
- 멀티스레드 환경에서의 동기화 문제를 직접 경험하며 lock의 중요성을 배웠습니다.
    - 여러 개의 스레드가 동시에 공유 자원에 접근하여 문제가 생겼습니다. 하지만 lock을 사용해서 공유 자원을 보호하는 방법으로 해결했습니다.
